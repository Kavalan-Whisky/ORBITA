# ORBITA

An interactive guide to the solar system. Calmly nerdy, scientifically grounded, visually distinctive.

## Files
- `index.html` — Hero with animated CSS solar system (8 orbits at different speeds, pulsing sun), mission control data bar, grid of 8 planet cards (each with unique CSS-rendered planet body), distance scale visualization, mission log of 11 selected missions
- `planet.html` — Detailed Jupiter page with large planet rendering (Great Red Spot, atmospheric bands), 8 data cells, essay sections on the storm, the four Galilean moons, mission history
- `styles.css` — Full design system with star-field background, nebula gradients, planet body gradients
- `main.js` — Planet card cursor-follow glow, scale dot tooltips, nav hide-on-scroll

## Aesthetic
- Deep space void + starlight + per-planet accent colors (gold sun, ice cyan, plasma orange)
- Fraunces (variable serif) + Space Grotesk + JetBrains Mono
- Animated orbits at scientifically-suggestive periods (faster for inner planets)
- All 8 planets are pure CSS — radial gradients + procedural Jupiter banding
- Procedural star field as background, double-layer with nebula gradient overlays
